Estás diseñando una aplicación web para subir ficheros al servidor. Haciendo uso deDOM, consigue que el usuario pueda adjuntar tantos ficheros como desee. Por tantoinicialmente será necesario un input de tipo fichero, un botón “Adjuntar otro fichero”y un segundo botón “Enviar”, de modo que al pulsar el primero de los botonesautomáticamente aparezca otro input para elegir fichero.
